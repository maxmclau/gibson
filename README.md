# Gibson

Tool used to build all Moa projects

#### Installation

```Shell
$ git clone git@github.com:moa/gibson.git
$ cd gibson
$ gibson --install
```
#### Update

```Shell
$ gibson --update
```
#### Build

```Shell
$ gibson --build -n atmega256rfr2 -a avr -t 256RFR2XPRO -f main.cpp
```
