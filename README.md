# RifRaf

Build tool for RifRaf's firmware and core files.

### Install
```Shell
git clone https://github.com/moa/rifraf-build.git
cd rifraf-build
./build.sh --build -n atmega256rfr2-a avr -t 256RFR2XPRO
```
