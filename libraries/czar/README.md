# Czar

[Atmel Lightweight Mesh](http://www.atmel.com/tools/lightweight_mesh.aspx) abstraction layer written in C++.

#### Install

```Shell
$ cd ~/Documents/Arduino/libraries
$ git clone https://github.com/moa/czar-lib.git
```

#### Usage

```Arduino
void setup() {
    Czar.setup();
}
    
void loop() {
    Czar.loop();
}
```
